# Elliott Assistants

A collection of Claude agents for personal productivity and task management.

## Agents

| Agent | Description |
|-------|-------------|
| [Calendar Management](/.claude/agents/calendar-management.md) | Organize, optimize, and manage calendar events and scheduling |
| [Email Triage](/.claude/agents/email-triage.md) | Process, prioritize, and draft responses to emails |
| [GitHub Repo Management](/.claude/agents/github-repo-management.md) | Maintain and organize GitHub repositories and workflows |
| [Meeting Prep](/.claude/agents/meeting-prep.md) | Prepare for meetings with agendas, talking points, and context |

## Usage

These agents are designed to work with [Claude Code](https://claude.com/claude-code) and can be invoked as custom agents.

### Quick Start

1. Clone this repository
2. Navigate to the directory in Claude Code
3. Use the agents by referencing them in your conversations

### Agent Structure

Each agent markdown file contains:
- **Capabilities**: What the agent can help with
- **Instructions**: How the agent approaches tasks
- **Example Tasks**: Sample prompts to get started
- **Templates**: Useful formats and structures

## Adding New Agents

To add a new agent:

1. Create a new markdown file in `/.claude/agents/`
2. Follow the existing agent structure:
   - Title and description
   - Capabilities section
   - Instructions section
   - Example tasks
   - Any relevant templates
